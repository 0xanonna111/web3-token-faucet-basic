# Web3 Token Faucet Basic

This project helps beginners learn how crypto faucets work.

You will learn:
- How a faucet smart contract works
- How to connect a wallet
- How to request test tokens
- How to show transaction status
- How Web3 apps send tokens

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-token-faucet-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- faucet.js = Faucet contract functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Request test tokens
- Show transaction hash
- Simple faucet UI
- Smart contract example

## Learning Goal
This project explains how test token faucets work.
